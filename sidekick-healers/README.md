# Sidekick Air - Healers-Inspired (Premium Wellness)

## Design 3: Premium Wellness Aesthetic

This prototype represents the **Healers-Inspired** design variation for Sidekick Air, targeting upscale wellness centers, premium clinics, and established practices.

## Design Philosophy

**Sophisticated Premium Wellness**

- **Visual Personality:** Refined, polished, elevated, natural
- **Color Strategy:** Deep Sage + Terracotta + Warm Cream (earth tones)
- **Typography:** DM Sans (display) + Inter (body) + Fraunces (serif for quotes)
- **Whitespace:** Generous (40-50% breathing room for premium feel)
- **Target Market:** Upscale positioning, established buyers valuing quality

## Setup Instructions

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The development server will start at `http://localhost:5173`

## Key Features

- Earth-tone color palette (Sage + Terracotta)
- Three-font system with serif accents
- Slower animations (0.8s for premium feel)
- Generous whitespace throughout
- Responsive design (mobile, tablet, desktop)
- All sections from design specification

## Technology Stack

- React 18
- Tailwind CSS 3

- React Icons
