# Sidekick Air - TherapyFlow Design Prototype

A complete React + Tailwind CSS prototype implementing the **TherapyFlow-Inspired** design variation for Sidekick Air's landing page.

## Design Philosophy

**Warm Healthcare Professional**

This design balances professional healthcare credibility with warm human connection, creating a trustworthy and approachable aesthetic ideal for conservative healthcare markets.

### Visual Identity

- **Color Palette:** Sage Green (#5FB5A5) + Electric Blue (#387CFF) + Warm Cream (#F5F3EF)
- **Typography:** DM Sans (display), Inter (body), Fraunces (accent serif)
- **Aesthetic:** Professional trust-building with emotional warmth
- **Target Audience:** Traditional therapists, sports medicine professionals, healthcare practitioners

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Opens at `http://localhost:5173`

### Build

```bash
npm run build
```
