# Sidekick Air - Design 2 (Xpedition-Inspired)

**Design Philosophy:** Modern Tech Innovation

A clean, conversion-optimized landing page prototype featuring a modern SaaS minimal aesthetic.

## Design Characteristics

- **95% Monochrome Palette:** White, gray, and near-black foundation with blue reserved exclusively for CTAs
- **Single Font:** Inter throughout for consistency and readability
- **Generous Whitespace:** 40-50% of the design is intentional empty space
- **Electric Steel Blue (#387CFF):** Used ONLY for call-to-action buttons and interactive elements
- **Clean SaaS Aesthetic:** Professional, tech-forward, conversion-optimized

## Tech Stack

- **React 18+** - Modern component-based architecture
- **Vite** - Fast build tool and dev server
- **Tailwind CSS 3+** - Utility-first styling
- **React Icons** - Icon library

## Setup Instructions

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will run at `http://localhost:5173`

## Features Implemented

### Sections

1. **Hero** - Full-screen impact with headline, CTAs, and product image
2. **Stats Bar** - Social proof numbers (500+ professionals, 800+ lbs, etc.)
3. **Features Grid** - 6 key features with icons
4. **Product Showcase** - Large product detail image with overlay text
5. **How It Works** - 3-step setup process
6. **Testimonials** - 3 professional reviews
7. **Sidekick Care App** - Dark section highlighting app integration
8. **Pricing CTA** - Conversion-focused pre-order section
9. **Footer** - Minimal navigation and copyright

### Interactions

- Fixed navigation with scroll effect
- Mobile hamburger menu
- Button hover states
- Card hover effects
- Smooth scroll to sections
- Responsive design (mobile, tablet, desktop)

**Color Palette:**
- Primary Blue: #387CFF

**Typography:**
